Online Ordering System Manual

IWCUID Insert the Ionian Weddings Client Unique Identifier.
Title Select the client's title from the predefined options in the dropdown.
First Name Insert the client's first name.
Last Name Insert the client's last name.
DOB This defines the new clients date of birth (DOB).
Mailing List This is selected as Yes or No depending on wether the new client wants to be on the mailing list or not.
Email Insert the client's email address.
Tel Insert the client's telephone (Tel) number.
Mob Insert the client's mobile (Mob) number.
Fax Insert the client's facsimilie (Fax) number.
Address 1 Insert the client's address (line 1).
Address 2 Insert the client's address (line 2).
Address 3 Insert the client's address (line 3).
Town Insert the client's town.
County Insert the client's county.
Country Select the client's country from the predefined options in the dropdown.
Postcode Insert the client's postcode.
Wedding Date Insert the client's wedding date.
Destination Select the client's desired destination from the predefined options in the select box.
